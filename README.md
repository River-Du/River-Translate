# 超级无敌好用的翻译软件

一个轻量的 Windows 桌面翻译工具。安装 Python 后无需额外依赖，适合日常查词、短句翻译、剪贴板翻译和多引擎对照使用。

## 主要功能

- 支持谷歌翻译、百度翻译、DeepL、自定义 AI 1、自定义 AI 2
- 默认使用谷歌免费接口，无需 API Key
- 支持自动检测源语言
- 支持输入停止后自动翻译
- 支持翻译完成后自动复制结果
- 支持剪贴板翻译：复制新文本后自动翻译
- 支持保存最近翻译历史并一键回填
- 支持窗口置顶，方便边看边翻译

## 快速开始

确保电脑已安装 Python 3.8 或更高版本。

双击 `run.bat` 即可启动。

也可以在命令行运行：

```bash
python src/main.py
```

## 基本使用

1. 在上方输入框输入要翻译的文本。
2. 选择源语言、目标语言和翻译引擎。
3. 点击“翻译”，或直接按 Enter。
4. 在结果区查看译文。

常用快捷键：

| 快捷键 | 功能 |
| --- | --- |
| Enter | 翻译 |
| Ctrl + Enter / Shift + Enter | 输入换行 |
| Escape | 清空输入和结果 |

## 支持语言

支持自动检测源语言，并可翻译中文、英语、日语、韩语、法语、德语、俄语和西班牙语。

## 翻译引擎说明

| 引擎 | 说明 | 是否需要配置 |
| --- | --- | --- |
| 谷歌翻译 | 默认使用免费接口，也可切换 Cloud API | 免费接口不需要；Cloud 模式需要 API Key |
| 百度翻译 | 使用百度通用翻译 API | 需要 AppID 和 SecretKey |
| DeepL | 支持 Free API 和 Pro API | 需要对应 API Key |
| 自定义 AI 1 / 2 | 兼容 OpenAI 格式接口 | 需要 API Key、Base URL 和模型名 |

谷歌免费接口是公共接口，优点是开箱即用；缺点是稳定性和翻译质量可能受网络、服务状态影响，在国内网络环境下可能不可用。如果需要更稳定的结果，建议配置 Google Cloud、DeepL、百度或自定义 AI 接口。

## 常用设置

点击“设置”可以配置各翻译引擎的 API 信息、接口类型和单次翻译最大字数。

底部开关说明：

- 自动翻译：输入停止约 1 秒后自动翻译。
- 自动复制：翻译完成后自动复制译文。
- 剪贴板翻译：复制其他软件中的文本后，自动填入并翻译。
- 窗口置顶：让窗口保持在其他窗口上方。

## 翻译历史

点击“历史”可以查看最近翻译记录，并将某条记录回填到主界面。

点击“清空历史”会立即清空全部历史记录。

## 隐私说明

程序会在 `user_data/` 目录保存本地配置和翻译历史。这里可能包含 API Key 和翻译文本，请不要随意分享这个目录。

翻译时，输入文本会发送给当前选择的翻译服务。请根据自己的隐私需求选择合适的引擎。

## 常见问题

### 双击 `run.bat` 没反应

通常是 Python 没有安装，或没有加入系统 PATH。可以先在命令行运行：

```bash
python --version
```

如果没有显示版本号，请重新安装 Python，并勾选“Add Python to PATH”。

### 谷歌免费翻译不可用或结果异常

谷歌免费接口不需要配置，但它不是正式付费 API，可能受网络环境和服务状态影响，在国内网络环境下也可能无法访问。可以稍后重试，或切换到其他已配置的翻译引擎。

### 为什么百度、DeepL、自定义 AI 不能直接用

这些接口需要你自己准备对应平台的 API Key 或账号信息。配置完成并保存后即可使用。
